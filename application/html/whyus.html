Why us?

Well, there are many reasons. Let us just get into them already to not waste your time.

Link previews

This is probably one of the most important reasons to use Scratchhh. If you just want to get a project across on Discord or Twitter (of all things), just paste the URL and be done.

Info embeds

While Scratch offers playable project embeds, they don't offer information, which is what I ended up adding. I think it is worth knowing the name of a project or how many views it has more than the actual project itself.

Consistent dark mode

Saves power and battery life and is easy on your eyes. More websites are adapting dark mode as time goes forward.

Project archive

Don't need the WayBack Machine when you got this! Quick explanation: Anytime a project is downloaded, the project is checked for differences. If that project is different from a cached project, it is stored. If not, then it just gets scrapped and it uses the same version. This is very useful in case you want some older version of a project because it has less bugs or didn't have an update which ruined the entire project. Anyway, you can access the archive using the /archive endpoint.
