About

scratchhh.xyz is a improved clone of scratch.mit.edu. It contains features one may find useful and are not available on Scratch. Also, it has a unique design so it doesn't look like another ModShare clone.

   scratchhh.xyz is made with the python3 "Flask" framework. In addition, it partially uses the scratchhh library for presenting information and the generator3 library to export blocks in Scratch 3 sprites to scratchblocks for download. That's just server stuff. Since project thumbnails take millions of years to load, I used lazysizes to lazy-load the images, hence, better performance. Projects are loaded with Turbowarp since Scratch's engine is much slower.

   Also did you know you can post block gists with this thing?
